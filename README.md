# sgTraffic

Traffic stats for spacegate

# install

```
npm install
grunt build
```

# running

```
node app/server.js
```

or use the ```run.sh``` script in the ```shell-scripts``` folder.

# rrd

Use the ```rrd/createDb.sh``` script.

1 Week every step (= every 10 seconds)
3 Years every 30th step (= every 5 minutes)
